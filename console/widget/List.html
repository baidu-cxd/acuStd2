# 表格

适用于大量数据的结构化呈现。用户可以通过表格控件添加内容，进行排序、筛选、折叠展开等操作。 通常，表格会根据父级页面定义宽度，或根据父级页面自适应宽度，因此本文只对表格内容设计和高度进行了定义。

# ② 单选/多选

表格可支持单选、多选：用户点击选择控件，选中某一列或多列内容后可进行后续操作。

# ③ 数据筛选及搜索

支持表头筛选、与其他控件组合筛选、搜索。（筛选排序是针对所有数据进行的，且会触发列表刷新）
