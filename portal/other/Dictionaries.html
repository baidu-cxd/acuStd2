# 规范字典

为了确保设计师，产品经理和程序员对规范有统一的认识，在此针对常见的重要名词进行阐述：

  • 速查表
  • 重要名词注释

# 速查表

中文名西文名描述
入口页portal百度云的官网，产品页，解决方案页等展示型页面
控制台console百度云的控制台页面
塔设计Tower Design百度云 portal 端设计规范的昵称
组件component页面的基本组成单元，如 按钮 ，输入表单
基础组件base-component最小的组件单元，如按钮，输入框
模块组件module-component由基础组件组合成的复杂组件，如 卡片，模态框
板块组件/楼层module-component组成页面的直接单元，比如由3个卡片并列组成的 三项卡片板块 因为我们的页面一般是由很多板块一层一层堆砌起来的，所以也可以称之为 楼层

# 重要名词注释

# 塔设计 (Tower Design)

百度云 portal 端页面在设计页面的时候，就像修塔一样，按板块（section）一层一层地堆砌，因此我们把我们的平台设计规范称为 “塔（Tower）”，我们的组件库名称为 Tower Design UI Kit
