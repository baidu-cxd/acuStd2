# 百度云官网 Portal 设计规范

百度云 Portal 端平台规范是针对百度云官网页面编写

>设计规范可以起到统一设计语言，提高风格辨识度的作用。一致的设计语言降低了产品的设计和开发成本，降低用户的学习成本；设计异质化可以更好的强化品牌属性。
