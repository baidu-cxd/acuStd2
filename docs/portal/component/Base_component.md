# 基础组件综述

基础组件是组成百度云 portal 端页面最基本的组成单元。一个基础组件的构成是根据产品的实际需求结合百度云视觉风格产生。
对于基础组件的样式，我们进行严格的定义，以确保百度云整体具有统一的视觉风格和用户体验。

百度云页面内的基础组件有：

- [按钮](/portal/component/button.html)
- [超链接](/portal/component/form.html)
- [表格](/portal/component/hyperlink.html)
