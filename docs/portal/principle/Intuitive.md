# 直观性

直观其实就是少猜，尽可能的减少用户的学习成本，用最简单的表叔形式传达给用户最直观的信息。页面中第一屏的位置是最重要的，所呈现的信息也应该是对直观性要求最高的

## 如何实现

### 1.文案传达简明扼要

针对页面整体风格，我们更倾向在文案上直观的传达更重要的信息以减少用户的浏览负担

### 2.图形化的传达形式

针对文案进行提炼总结，更多的用图形化代替长篇话术文本，直观的传达给用户最想看到的信息，在整体上提升了页面美观度的同时也能够减少阅读成本
