# 占位（Placeholder）

占位是为了在没有数据时先占据一个固定的位置，等有内容之后进行替换，保证了页面的样式统一且不会发生页面跳动。

 [[toc]]

### 占位符

百度云console的占位符为英文的破折号 “-”，如下图所示。

![图片](http://baiduyun-guideline.bj.bcebos.com/console/widget/placeholder/01_2x.png)
