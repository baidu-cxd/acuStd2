# 步进器（Stepper）

步进器是一个只定义数目的控件，该控件既支持用户通过点击加减按钮来修改数目，也支持用户直接手动录入。

<mark>步进器初始状态一般为1，根据业务逻辑的差异设置特定的上限值。</mark>
