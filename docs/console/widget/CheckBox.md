# 复选框（Check Box）

多用于表单、列表中，是基于文本的控件，允许一个或更多选择项，通过单击做标记，再次单击取消标记，可学性良好；传统上采用方形代表复选框，根据所对应的文本内容决定排列形式；
主要有三种选择状态：<mark>未勾选、已勾选、半勾选；</mark>
