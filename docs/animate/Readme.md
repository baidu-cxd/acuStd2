# 动画概述

动画是有效提高页面体验的设计形式，百度云体系下的动画主要是以状态过渡的交互动画，和提高品质感的细节动画两种形式为主。主要的目的在于:

>1. 帮助用户理解页面交互形式
>2. 提高页面视觉效果，加深用户对页面的印象，降低长时间浏览的枯燥感

## 开始设计动画：

我们将常见的动画形式做了预设，如果需要使用某种动画，可以查阅     [基础动画](/animate/Base.html)
